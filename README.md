# Киберқауіпсіздік негіздері (Сайттың аты)

Бұл — ақпараттық қауіпсіздік және кибергигиена ережелерін үйретуге арналған оқу жобасы.

## 🚀 Жобаның мақсаты
Пайдаланушыларға интернеттегі қауіпсіздік негіздерін бейнесабақтар мен интерактивті тесттер арқылы түсіндіру.

## 🛠 Қолданылған технологиялар
* **HTML5** — сайттың құрылымы.
* **Tailwind CSS** — заманауи дизайн және стильдер.
* **JavaScript** — тест тапсырмаларының логикасы.

## 📺 Видеосабақтар
Жобада келесі тақырыптар бойынша видеолар бар:
1. Ақпараттық қауіпсіздік негіздері.
2. Қорғану әдістері.
